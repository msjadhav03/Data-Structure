# Recursion

- More Memory
- Recursion Stack
- Infinite call to recursion makes stack memory out off

1. Tree and Graph Traversal
2. Sorthing Algorithm
3. Divide and Conquer
4. Fractal Generation
5. Backtracking Algorithm
6. Memoization - Caching result of excessive recursive call
